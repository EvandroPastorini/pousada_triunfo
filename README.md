# Pousada Triunfo

Site institucional moderno para a Pousada Triunfo, criado com Next.js, TypeScript e Tailwind CSS.

## Rodando localmente

```bash
npm install
npm run dev
```

Abra `http://127.0.0.1:3000`.

## Build de produção

```bash
npm run build
npm start
```

## Deploy na Vercel

O projeto está pronto para deploy na Vercel usando as configurações padrão de Next.js.

## Conteúdo editável

Textos, contatos, navegação, galeria e acomodações ficam em:

```text
src/lib/site-data.ts
```
